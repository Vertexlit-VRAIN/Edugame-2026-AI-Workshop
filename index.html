"AI has been used in many fields and has become very popular. It can do many things like writing, coding, and analysis. Some people think it is good and some think it is bad. In education, students are using it more and more. This shows that AI is important."

1. Identify the main claim, if any
2. List two specific weaknesses in the argumentation
3. Suggest a concrete revision that strengthens the argument
4. Rate its argumentative strength (1–5) and justify

Implications & Discussion

What Changes — What to Discuss

01 — Specify
Define the goal clearly enough that an agent can pursue it. This is a design skill that can be taught.
02 — Supervise
Intervene with domain knowledge the agent cannot replicate. Identify when it is confident and wrong.
03 — Review
Evaluate output against standards the agent does not hold. Faculty becomes quality assurance, not execution.

"If an agent can complete an entire course — what is learning for?"
